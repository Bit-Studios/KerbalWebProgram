What is Kerbal Web Program?

Kerbal Web Program is a ksp2 mod that gives you tools to use web content in the game.

  • Included web server
  • Automatic documentation for apis
  • User extendable api, with custom api extentions and mods
  • Button api
  • Ingame browser

So what can I do with this mod?

This mod was created so you could become mission control. You can fully automate missions, Create web interfaces, Use any programming language you like And Expand the apis with your own or use community created ones.
Some examples of what could be created with this mod

  • Craft performance tracker
  • Mission control interface
  • Youtube and Twitch intergration
  • Dv calculator
  • Ingame Scott Manley tutorials
